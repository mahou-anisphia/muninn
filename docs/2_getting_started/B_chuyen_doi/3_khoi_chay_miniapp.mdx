# Khởi chạy Miniapp

Sau khi đã chuẩn bị project SPA, bước tiếp theo là khởi chạy và preview trên thiết bị.

## Bước 1: Khởi động Development Server

Mở terminal trong thư mục project và chạy:

```bash title="Terminal"
npm run start
```

Đợi đến khi development server khởi động thành công. Thông thường sẽ hiển thị URL dạng:

```
Local:   http://localhost:3000
```

:::tip Kiểm tra app trước
Trước khi preview trên Android, hãy mở URL trên browser để đảm bảo app chạy đúng. Nếu có lỗi, sửa trước khi tiếp tục.
:::

# Motion stopGyroscope - Dừng gyroscope

Tắt cảm biến gyroscope.

## API Call

```javascript
window.WindVane.call('WVMotion', 'stopGyroscope', {}, () => {
  console.log('Gyroscope stopped');
});
```

## API liên quan

- [startGyroscope](./motion_startGyroscope) - Bật gyroscope

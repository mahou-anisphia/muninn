# Bước tiếp theo

Bạn đã chạy thành công miniapp đầu tiên. Dưới đây là các hướng đi tiếp theo.

## Sample code với đầy đủ Bridge API

Chúng tôi cung cấp một miniapp mẫu demo **toàn bộ Bridge API (jsAPI)**.

Clone repo sample code về máy:

```bash title="Terminal"
git clone https://github.com/mahou-anisphia/miniapp-sample-code.git
```

Di chuyển vào thư mục project:

```bash title="Terminal"
cd miniapp-sample-code
```

Cài đặt dependencies:

```bash title="Terminal"
npm install
```

Sau đó chạy preview theo hướng dẫn ở [Phát triển Miniapp](./phat_trien_miniapp).

:::caution API đặc biệt
`getAuthCode` là API riêng của Tammi Superapp, **không hoạt động** ở chế độ local preview. API này chỉ hoạt động sau khi deploy lên nền tảng Tammi.
:::
